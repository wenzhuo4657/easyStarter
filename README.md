# spring-boot-starter

该项目依托与springboot的自动装配机制，适配版本为springboot3.0之后的imports文件。
详情可见: [SpringBoot 3 替换 spring.factories](https://www.cnblogs.com/jingzh/p/18793407)



| 名称            | 功能       |  
|---------------|----------|
| DoRateLimiter | 限流       |
| hystrix       | 熔断       |
 | methodext     | 方法返回值的拦截 |
|whiteList| 方法的白名单放行 |
